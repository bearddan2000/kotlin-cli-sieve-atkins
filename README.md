# kotlin-cli-sieve-atkins

## Description
Given a number N, print all prime numbers smaller than N.

## Tech stack
- kotlin

## Docker stack
- gradle:4.2.1-jdk8-alpine

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
